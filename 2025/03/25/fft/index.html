复数

首先，我们需要重新定义开根号运算，开 nnn 次根号一定 nnn 个根。

−1=±i\sqrt{-1} = \pm \text{i}−1​=±i，即虚数单位。

复数：a+bia + bia+bi。

复平面：实数轴与垂直于它的虚数轴所构成的平面直角坐标系，包含了所有的复数，(a+bi)→(a,b)(a + bi) \to (a, b)(a+bi)→(a,b)。

复数的模长：z=a2+b2z = \sqrt{a^2 + b^2}z=a2+b2​，z⩾0z \geqslant 0z⩾0。

复数的幅角：θ=arctan⁡ba\theta = \arctan\tfrac{b}{a}θ=arctanab​。

即：a+bi=zexp⁡θia + bi = z \exp \theta ia+bi=zexpθi。

复数加/减法：(a+bi)+(c+di)=(a+b)+(c+d)i(a+bi)+(c+di) = (a+b)+(c+d)i(a+bi)+(c+di)=(a+b)+(c+d)i。

复数乘法：(a+bi)(c+di)=(ac−bd)+(bc+ad)i(a+bi)(c+di) = (ac - bd) + (bc + ad)i(a+bi)(c+di)=(ac−bd)+(bc+ad)i。

复数乘法（极坐标）: z1exp⁡θ1i×z2exp⁡θ2i=(z1z2)exp⁡(θ1+θ2)iz_1\exp\theta_1 i \times z_2\exp\theta_2i = (z_1z_2)\exp(\theta_1+\theta_2)iz1​expθ1​i×z2​expθ2​i=(z1​z2​)exp(θ1​+θ2​)i。

共轭复数：a+bia + bia+bi 和 a−bia - bia−bi 共轭，相乘幅角抵消，只剩实部。
